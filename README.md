## Features

[서비스]
- 소셜(구글) 로그인 / 로그아웃
- 메인 상품리스트(이미지 슬라이더)
- 상품 구매하기(옵션 선택)
- 상품 결제(배송지 정보 입력)
- 마이페이지(주문 조회)

[어드민]
- 상품의 옵션을 선택하여 셀러 상품 등록
- 위즈윅 에디터(CKEditor4)
- 이미지 등록
- 날짜와 시간, 검색 조건을 통한 필터링
- 페이지네이션
- 리스트 순서 <-> 역순 필터링
